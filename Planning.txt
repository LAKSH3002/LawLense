Researching about constitution, laws, articles, cases, judgements
Collecting data from multiple platforms (web scraping, APIs, Databases)

Take a user query
Convert query → embedding
Search in FAISS
Collect retrieved text
Feed them to an LLM
Generate an answer

Industry Pipeline to generate an answer based on a query
Query → SentenceTransformer → FAISS → Retrieved articles → Gemini/Ollama → Answer
